# 现有主流网格策略分析（含风险）

## 一、网格策略的统一抽象（先给你一把总钥匙）

无论名字多花哨，所有网格策略本质都是：

> 在价格区间内，用规则化的挂单，将价格波动转化为路径收益（realized PnL），并在区间失效时承担方向性风险。

区别只在三件事：

- 是否带方向偏置（多 / 空 / 中性）
- 库存是否有上限
- 区间失效后怎么处理

## 二、主流网格类型拆解

### ① 中性网格（Neutral Grid）

#### 结构定义

- 当前价格为分界线
- 上方：Sell Limit
- 下方：Buy Limit
- Buy / Sell 分界线随价格移动

#### 收益来源

纯粹来自：

- `buy at Li → sell at Li-1`
- 不依赖方向判断

#### 核心优势

- 震荡市效率最高
- 不预测方向
- 策略逻辑最干净

#### 核心风险

- **趋势风险（致命）**
  - 单边上涨 → 空头库存无限累积
  - 单边下跌 → 多头库存无限累积
- 库存无上限
- 本质是"隐含卖波动"
- 第一单 + re-anchor 决策
  - 错一次，可能毁整段行情

#### 常见误区

- 以为"中性 = 低风险"
- 忽视最大仓位和极端路径

#### 适用行情

- 明确区间震荡
- 波动充足但无趋势

### ② 做多网格（Long-biased Grid）

#### 结构定义

- 只允许正库存
- 下跌 → Buy（加仓）
- 上涨 → Sell（减仓）
- 通常需要底仓

#### 收益来源

- 网格价差收益
- 底仓随行情上涨的 Beta

#### 核心优势

- 适合现货
- 心理压力相对小
- 与长期看多逻辑兼容

#### 核心风险

- **趋势下行风险**
  - 下跌中不断加仓
  - 浮亏和资金占用快速扩大
- **区间失效**
  - 跌破下边界后变成"死扛多头"
- 过度密集网格
  - 手续费侵蚀

#### 常见误区

- 把它当"低风险定投"
- 不设置最大仓位
- 用高杠杆跑做多网格

#### 适用行情

- 中长期偏多
- 有反复回撤的上行行情

### ③ 做空网格（Short-biased Grid）

#### 结构定义

- 只允许负库存
- 上涨 → Sell（加空）
- 下跌 → Buy（回补）

#### 收益来源

- 网格价差
- 下跌趋势中的 Beta

#### 核心优势

- 可在高位震荡或缓慢下行中赚钱
- 可作为组合对冲工具

#### 核心风险（最危险）

- **无限风险**
  - 上涨无上限
  - short squeeze
  - 波动集中向上
- 保证金 / 强平
- 非线性死亡

#### 常见误区

- 把它当"做多网格反向"
- 不设硬上限
- 忽略 funding / 借币成本

#### 适用行情

- 明显高位震荡
- 下行趋势早期（需极强风控）

### ④ 区间固定网格（Static Range Grid）

#### 结构定义

- Buy / Sell 层级固定
- 不随价格重心移动

#### 收益来源

- 被动吃单
- 类似被动做市

#### 核心风险

- 一旦价格偏离区间
- 策略失效，库存单边暴露

#### 本质

- 不是状态机
- 是一次性结构

### ⑤ 趋势网格（Trend-following Grid）

#### 结构定义

- 只允许顺趋势方向开仓
- 逆趋势方向冻结

#### 收益来源

- 网格价差
- 趋势方向 Beta

#### 核心风险

- 趋势判断错误
- 变成"高频追涨杀跌"

## 三、风险的统一本质（非常重要）

你可以把所有网格风险，统一成一句话：

> **网格不是在赌方向，而是在赌"价格会继续在你允许的区间内来回"。**

所有爆仓 / 大亏，几乎都来自：

- 区间假设被破坏
- 库存上限不存在
- 极端行情中没有退出机制

## 四、为什么市面 90% 网格"长期不行"

### 1️⃣ 卖的是"稳定感"，不是"风险管理"

- 强调成交次数
- 回避最大仓位、极端路径

### 2️⃣ 隐含卖波动

- 网格 ≈ short gamma
- 大行情必然被打穿

### 3️⃣ 产品层面掩盖风险

- UI 不展示最大可能仓位
- 不展示强平路径

## 五、交易员级判断清单（你以后可以直接用）

看到任何网格策略，先问 6 个问题：

1. **最大仓位是多少？**
2. **价格单边走时会发生什么？**
3. **区间失效后是退出、冻结，还是死扛？**
4. **第一单和 re-anchor 谁决定？**
5. **手续费是否被真实计入？**
6. **这是卖波动，还是卖方向？**

> 回答不清楚的，直接判死刑。

## 六、一句话总结（给你定心锚）

> **网格不是"稳赚工具"，而是一种需要极强边界管理的"库存型策略"。**

你现在已经具备一个能力：把"网格"从营销话术里剥离出来，只看它的真实风险敞口。
